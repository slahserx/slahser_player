# Slahser Player 0.8.0 更新日志

发布日期: 2025年3月16日

## 更新内容

- 新增播放列表功能，支持创建、删除和编辑播放列表
- 优化音量控制，修复切换歌曲时音量滑块动画问题
- 改进用户界面，优化布局和响应式设计
- 新增迷你播放器模式，可在任务栏显示播放控制
- 添加更多音频格式支持，包括FLAC和OGG
- 修复多个已知BUG和稳定性问题

## 安装说明

### 安装版 (.exe)

1. 双击下载的 .exe 文件
2. 按照安装向导完成安装
3. 安装完成后，可以从开始菜单或桌面快捷方式启动应用

### 便携版 (.zip)

1. 下载 .zip 文件
2. 解压到任意文件夹
3. 运行 frontend.exe
